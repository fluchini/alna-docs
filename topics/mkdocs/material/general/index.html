    Guia Geral Markdown

    Toda a documentação é feita utilizando a ferramente MkDocs. Para mais detalhes sobre o MkDocs visite mkdocs.org.

    Para criar esta documentação utilizamos o Markdown. O Markdown é uma técnica mais simplificada e eficiente de fazer documentação, voltada geralmente para times técnico da Tecnologia da Informação.

    Para saber mais sobre o Markdown visite o site Markdown Wikipedia.

    Além do guia rápido abaixo, também indicamos os seguintes guias para você consultar tudo o que você pode fazer com o Markdown.

    • Guia de Referênca Geral: Markdown Guide
    • Ferramenta de Uso Online: Stackedit.io
    • Guia de Referênca Material: Material Guide

    Lista\Tópicos

    Listas podem ser criadas com enumeração arábica ou simplesmente na forma de tópicos.

    • Sed sagittis eleifend rutrum
    • Donec vitae suscipit est
    • Nulla tempor lobortis orci
    ### Por Tópicos.

* Sed sagittis eleifend rutrum
* Donec vitae suscipit est
* Nulla tempor lobortis orci
    1. Sed sagittis eleifend rutrum
    2. Donec vitae suscipit est
    3. Nulla tempor lobortis orci
    ### Por Enumeração.

1. Sed sagittis eleifend rutrum
2. Donec vitae suscipit est
3. Nulla tempor lobortis orci
