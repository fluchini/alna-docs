# Abas

Podemos criar abas para agrupar conteúdos.

Exemplo de ciração de agrupadores\abas para seu conteúdo:

=== "Unordered list"

    * Sed sagittis eleifend rutrum
    * Donec vitae suscipit est
    * Nulla tempor lobortis orci

=== "Ordered list"

    1. Sed sagittis eleifend rutrum
    2. Donec vitae suscipit est
    3. Nulla tempor lobortis orci

Côdigo Markdow:

``` md
### Exemplo de Abas

=== "Unordered list"

    * Sed sagittis eleifend rutrum

    * Nulla tempor lobortis orci

=== "Ordered list"
```
